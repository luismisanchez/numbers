# Quickstart: Simulation Lab

The Simulation Lab allows you to validate the EuroDreams Statistical Engine using Monte Carlo methods and historical backtesting.

## Running a Backtest

1. Navigate to the **Simulation Lab** tab.
2. Select a **Target Date** from the calendar.
3. Click **Run Backtest**.
4. The system will:
   - Filter history up to that date.
   - Re-calculate weights.
   - Show where the actual winner landed in the bell curve of 1M+ combinations.

## Running a Monte Carlo Simulation

1. Set the **Sample Size** (default: 10,000).
2. Choose **Hybrid** mode to compare purely random vs. engine-weighted logic.
3. Click **Start Simulation**.
4. Observe the "Statistical Shift" on the histogram. The "Realistic Zone" (25-35 pts) should align with the historical distribution peak.

## Responsible Gaming Note

All simulations are for **educational purposes only**. Matching a high score in the simulator does not translate to real-world probability. Always play responsibly.
